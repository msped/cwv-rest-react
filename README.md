### Sales

`GET /api/sales/`
- Returns all vehicles.

`GET /api/sales/:id/`
- Returns vehicle with supplied id.

`GET /api/sales/state/:id/`
- Returns the current sales state of a vehicle.

`POST /api/sales/reserve/:id/`
- Sends post data to reserve a vehicle.
